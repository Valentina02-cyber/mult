Режим случайных примеров


Данный режим нацелен на пользователей, которые уже знают таблицу умножения, так как является сложным для новичка. Поэтому мы рекомендуем начинать именно с таблицы на самые простые числа. Чтобы потренироваться в этом режиме, нужно:

1. Переключить на этот режим;

2. Тренироваться, кликая кнопку "Проверить", также нажимать на эту кнопку можно с клавиатуры нажатием на "ENTER";

3. По завершении тренировки, рекомендуется кликнуть кнопку "Сохранить", чтобы сохранить текущий процесс.
