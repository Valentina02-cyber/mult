Режим примеров в определенном диапазоне


Данный режим нацелен на повторение изученного материала, но ограничен только знаниями пользователя. Обучение проходит таким образом:

1. Переключить на этот режим;

2. Выбрать цифру из выпадающего списка;

3. Тренироваться, кликая кнопку "Проверить", также нажимать на эту кнопку можно с клавиатуры нажатием на "ENTER";

3. По завершении тренировки, рекомендуется кликнуть кнопку "Сохранить", чтобы сохранить текущий процесс.
